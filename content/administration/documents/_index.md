Ondsel Lens stores legal documents in a database and keeps all revisions so that it's possible to keep tracks which terms and conditions each user agreed to.

The following documents can be edited:

- [Privacy policy](privacy-policy)
- [Terms of service](terms-of-service)
- [Survey prompt](survey-prompt)
